# DISA STIG Checklists

Pre-filled STIG checklists documenting how the SRE platform satisfies each STIG requirement.

## Checklists

| STIG | Applied By |
|------|------------|
| Rocky Linux 9 (RHEL 9 STIG) | Ansible `os-hardening` role |
| RKE2 Kubernetes STIG | RKE2 default CIS profile + `rke2-hardening` role |
| Istio STIG | Manual configuration documented here |

## Format

Checklists are provided in `.ckl` format (DISA STIG Viewer compatible) and as YAML summaries for automated processing.
