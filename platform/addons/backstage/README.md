# Backstage Developer Portal (Optional Addon)

Backstage provides a unified developer experience with a software catalog, scaffolding templates, and integrated documentation.

## What It Does

- Software catalog listing all deployed services
- Software Templates for scaffolding new services (creates repo, Helm values, Flux config)
- TechDocs for team documentation
- Integration with Harbor, Grafana, and ArgoCD

## Dependencies

- Depends on: Istio, Keycloak (for SSO), Monitoring (for Grafana links)
